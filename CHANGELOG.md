# 更新日志

米家智能提醒插件的所有版本更新记录。

## [1.0.0] - 2024-03-20

### 🎉 首次发布

#### ✨ 新功能
- **插件架构**：完整的 Codebuddy 插件架构
- **5个核心命令**：
  - `/mijia-setup` - 配置米家账号和设备
  - `/mijia-add` - 添加定时提醒
  - `/mijia-list` - 查看提醒列表
  - `/mijia-status` - 检查设备状态
  - `/mijia-broadcast` - 手动播报提醒
- **智能助手**：对话式交互的 mijia-assistant
- **自动化任务**：每分钟检查提醒的自动化任务
- **完整技能**：基于现有 Skill 的完整功能

#### 🏗️ 技术特性
- **插件配置**：完整的 plugin.json 配置文件
- **路径支持**：`${CODEBUDDY_PLUGIN_ROOT}` 变量支持
- **脚本优化**：所有脚本支持插件环境
- **数据管理**：本地数据存储和备份
- **日志系统**：完整的操作日志记录

#### 📦 安装与部署
- **一键安装**：`install.sh` 安装脚本
- **安全卸载**：`uninstall.sh` 卸载脚本（含数据备份）
- **多平台支持**：macOS 和 Linux 系统
- **双版本支持**：WorkBuddy 和 Codebuddy 兼容

#### 🎨 用户体验
- **双主题图标**：深色/浅色主题图标
- **完整文档**：README.md 和配置指南
- **自然语言**：智能助手支持自然语言交互
- **错误处理**：完善的错误提示和恢复机制

#### 🔒 安全特性
- **本地存储**：所有数据本地加密存储
- **权限控制**：明确的工具权限配置
- **隐私保护**：不存储用户密码
- **安全备份**：卸载时自动备份用户数据

#### 📋 文件结构
```
mijia-smart-reminder-plugin/
├── plugin.json              # 插件配置文件
├── README.md               # 说明文档
├── CHANGELOG.md           # 更新日志
├── commands/              # 5个命令定义
├── skills/                # 完整技能实现
├── agents/                # 智能助手
├── automations/           # 自动化任务
├── scripts/               # 安装/卸载脚本
├── assets/                # 图标资源
└── data/                  # 用户数据（运行时生成）
```

### 🐛 已知问题
- 米家 API 连接为模拟实现，需要实际 API 集成
- 部分高级功能（如语音识别）为未来版本规划
- 需要用户安装 jq 工具以获得最佳体验

### 📝 注意事项
- 首次使用需要运行 `/mijia-setup` 配置账号
- 建议安装 jq 工具用于 JSON 处理
- 插件需要重启 WorkBuddy/Codebuddy 才能生效

## 版本号规则

本插件遵循 [语义化版本 2.0.0](https://semver.org/lang/zh-CN/)：

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正
